# RESOURCES.md — MRCEM Primary Exam Preparation

## Tier 1: College-Endorsed (Must-Have)

### RCEMLearning
- **What:** Online learning platform, included with RCEM membership
- **Why:** The only preparation material endorsed by RCEM. Basic Sciences module maps directly to the curriculum.
- **Access:** Automatically available once RCEM membership is active
- **Cost:** Included with membership (£200-300/year for UK members)

### RCEM Basic Sciences Curriculum (June 2010)
- **What:** The official syllabus document — every exam question maps to this
- **Access:** Available on RCEM website (members' area)
- **Status:** NOT YET OBTAINED — need to locate and save

## Tier 2: Proven Textbooks & Question Banks

### Robbins Basic Pathology (10th or 11th Edition)
- **What:** The gold-standard pathology textbook
- **Why:** Examiners write from this. Chapters 1-7 cover the core basic sciences.
- **Cost:** ~£50-60 (new), ~£30 (used)
- **Status:** TO ACQUIRE
